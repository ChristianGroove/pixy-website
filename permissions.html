Reviewer Instructions

Please follow these steps to test the OAuth flow and verify permission usage.

1

Go to Login Page

Navigate to app.pixy.com.co (Production Environment).

2

Login with Meta

Click the "Sign in with Meta Strategy" button on the login screen.

3

Complete OAuth Flow

Accept the requested permissions in the Facebook popup window.

  • pages_show_list
  • instagram_basic
  • pages_manage_metadata
  • whatsapp_business_messaging
  • whatsapp_business_management
  • whatsapp_business_manage_events
4

Verify Page Listing

Once redirected, you will see a list of Facebook Pages you administer. This demonstrates pages_show_list usage.

5

Connect Instagram

Select a page with an Instagram Business account connected. The dashboard will display the IG profile info. This demonstrates instagram_basic usage.

If you need a specific test user account, please use the credentials provided in the App Review submission form notes.
